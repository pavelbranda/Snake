# Snake Game

**Welcome to Snake!**
A simple Snake game built with JavaScript, HTML, and CSS.

- Move the snake using arrow keys.
- Eat food to grow.
- Avoid hitting walls or yourself.

## Features
- Classic snake gameplay
- Customizable grid size
- Scoring system
